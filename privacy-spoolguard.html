Privacy Policy – SpoolGuard (CZ + EN)

English

Effective date: March 16, 2026

1. Overview

SpoolGuard is a macOS app developed by Mast Dev Studio for local 3D filament inventory and print history tracking.

2. Data We Process

SpoolGuard does not collect or transmit personal user data.

3. Local Storage

User data (for example filament entries and print logs) is stored only on the user’s device.

4. Internet Connection

The app works fully offline and does not require an online account.

Česky

Datum účinnosti: 16. března 2026

1. Přehled

SpoolGuard je aplikace pro macOS od Mast Dev Studio, určená pro lokální správu 3D filamentů a historie tisků.

2. Jaká data zpracováváme

SpoolGuard nesbírá ani neodesílá osobní údaje uživatelů.

3. Lokální ukládání

Uživatelská data (např. seznam cívek, tiskové záznamy) jsou ukládána pouze lokálně na zařízení uživatele.

4. Připojení k internetu

Aplikace funguje plně offline a nevyžaduje uživatelský účet.
